Multiplication in Scratch!

Learn how to multiply two numbers using Scratch — step by step, like a pro! 🚀

⭐ We will make a program that asks for 2 numbers and shows their product!  |  🧮  Number1 × Number2 = Result
📦 Getting Ready
1

Create 3 Variables 📦

Variables → Make a Variable

Before we start coding, we need to create 3 boxes (called variables) to hold our numbers. Think of variables like labelled jars — we'll put numbers inside them!

  • 1 Click Variables on the left sidebar (the orange section).
  • 2 Click "Make a Variable" button — a popup appears.
  • 3 Type Number1 and click OK.
  • 4 Repeat: Make another variable called Number2.
  • 5 Repeat again: Make one more called Result.
📦 Create "Number1"
📦 Create "Number2"
📦 Create "Result"
💡 You should now see Number1, Number2, and Result appear in the top-right corner of the stage. These are your three jars — ready to hold numbers!
🚀 Starting the Program
2

Add the Start Block 🚩

Events → when 🚩 clicked

Every Scratch program needs a starting point. We use the 🚩 when clicked block — it runs all the code below it when you press the green flag!

  • 1 Click Events on the left sidebar (yellow section).
  • 2 Drag when 🚩 clicked onto the empty coding area (the white dotted space).
🚩 The start block is in the coding area!
⚡ The green flag is the play button of Scratch. Whenever you click it, your program starts running from this block!
💬 Asking for Numbers
3

Ask for the First Number 💬

Sensing → ask [ ] and wait

Now let's make the Scratch Cat ask a question! We use the ask [ ] and wait block — it shows a question and waits for the user to type an answer.

  • 1 Click Sensing on the sidebar (light blue section).
  • 2 Drag ask [What's your name?] and wait and snap it below the flag block.
  • 3 Click the white box inside the block and type: Enter First Number -
💬 The cat will ask "Enter First Number -"
🐱 Whatever the user types in response will be stored temporarily in a special block called answer. We'll use that next!
4

Save the Answer into Number1 💾

Variables + Operators + Sensing combined!

The user typed a number — now we store it in Number1. We use a set [Number1] to block combined with a × multiply operator and the answer block.

  • 1 Click Variables → drag set [my variable] to [0] and snap it below the ask block.
  • 2 Click the dropdown on the block and choose Number1.
  • 3 Click Operators → drag the green [ ] × [ ] (multiply) block into the value slot.
  • 4 Click Sensing → drag the answer block into the left slot of the multiply block.
  • 5 Type 1 in the right slot (multiplying by 1 converts the answer to a number!).
📋 Select "Number1" from dropdown
✖️ Add multiply block from Operators
🔵 Drop "answer" into left slot
1️⃣ Type "1" in the right slot
🔧 Why multiply by 1? Because the answer block stores text. Multiplying by 1 is a clever trick to turn it into an actual number that Scratch can calculate with!
5

Ask for the Second Number 💬

Sensing → ask [ ] and wait (again!)

Great job! Now we need to ask for the second number. It's the same idea as before — just a different question!

  • 1 Click Sensing → drag another ask [ ] and wait and snap it below the "set Number1" block.
  • 2 Type: Enter Second Number in the white box.
💬 Add "Enter Second Number" ask block
👀 Notice that your script is growing! Each block you add goes directly below the previous one — like building with LEGO bricks.
6

Save the Answer into Number2 💾

Same trick — this time for Number2!

Just like we did for Number1, we now save the user's second answer into the Number2 variable. Same steps, different variable!

  • 1 From Variables → drag set [my variable] to [0] below the second ask block.
  • 2 Click the dropdown → choose Number2.
  • 3 From Operators → drag [ ] × [ ] into the value slot.
  • 4 From Sensing → drop answer in the left slot.
  • 5 Type 1 in the right slot.
📋 Select "Number2" from dropdown
🔢 "set Number2 to 0" added
✖️ Add multiply block
🔵 Drop "answer" in left slot
1️⃣ Type "1" in the right slot
🎯 Your script now has 5 blocks stacked together! It asks for two numbers and saves them properly. We're almost done!
🧮 Calculating the Answer
7

Multiply! Set Result = Number1 × Number2 🧮

Variables + Operators = Magic!

Here's the exciting part — the actual multiplication! We set the Result variable to Number1 times Number2. Scratch will calculate this automatically! 🎉

  • 1 From Variables → drag set [my variable] to [0] and snap it at the bottom of the script.
  • 2 Click the dropdown → choose Result.
  • 3 From Operators → drag the [ ] × [ ] multiply block into the value slot.
  • 4 Click Variables → drag the oval-shaped Number1 block into the left slot of the multiply.
  • 5 Drag the Number2 oval block into the right slot.
📋 Select "Result" from dropdown
✖️ Multiply block placed in Result
🟠 Number1 dropped in left slot
✅ Result = Number1 × Number2 done!
🎉 The multiplication logic is complete! The Result variable will automatically show the correct answer when the program runs.
📣 Announcing the Answer
8

Make the Cat Say the Answer! 🐱💬

Looks → say + Operators → join

The calculation is done — but we want the Scratch Cat to announce the answer out loud! We combine a say [ ] block with a join [ ] [ ] block to display a nice message like "Answer = 2000".

  • 1 Click Looks on the sidebar (purple section) → drag say [Hello!] and snap it at the very bottom of the script.
  • 2 Click Operators → drag the green join [apple] [banana] block into the white box of the say block (replacing "Hello!").
  • 3 In the left slot of the join block, type: Answer = (with a space at the end!).
  • 4 Click Variables → drag the Result oval block into the right slot of the join.
🟣 Add "say Hello!" from Looks
🟢 Drop join block into say
✏️ Type "Answer = " in left slot
🟠 Drop "Result" in right slot
🔗 The join block is like glue — it sticks two things together! "Answer = " + Result becomes one message like "Answer = 2000". The cat will say it in a speech bubble!
🚀 Run It! See It Work!
▶

Click the Green Flag & Test! 🎯

Your Multiplication Machine is alive!

Your program is complete! Click the green flag 🚩 to run it. Watch what happens step by step on the right side of Scratch:

  • 1 The cat asks "Enter First Number -" — type any number and press Enter. (e.g. 40)
  • 2 The cat asks "Enter Second Number" — type the second number. (e.g. 50)
  • 3 The cat shows a speech bubble: "Answer = 2000" 🎉
  • 4 In the top-right corner you can also see Number1 = 40, Number2 = 50, Result = 2000 updating live!
💬 Cat asks for First Number!
🔢 Number1 = 40, now asking for 2nd!
🎉 Cat says "Answer = 2000"!
